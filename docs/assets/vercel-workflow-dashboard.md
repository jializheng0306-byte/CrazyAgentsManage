## 设计元素分析

| 元素 | 说明 | 本仪表板适配 |
|------|------|-------------|
| 顶部状态栏 | 工作流名称 + 运行ID + 状态圆点（绿色=完成） | 智能体名称 + 来源 + 状态 |
| 元数据行 | Created / Completed / Duration / Expiry / Storage | 开始时间 / 完成时间 / 耗时 / Token / 存储 |
| 标签页 | Trace / Events / Streams | Trace / Events（简化） |
| 搜索框 | 搜索 spans | 搜索任务/智能体 |
| 时间轴刻度 | 0, 10s, 20s, 30s, 40s, 50s, 1m, 1m10s | 相同，自适应 |
| 任务条 | 蓝色横条，右侧显示耗时标签 | 按角色着色，右侧耗时 |
| 嵌套缩进 | 竖线连接 + 缩进表示层级 | 相同，最多 3 层 |
| 子跨度 | 灰色条纹 + 蓝色/橙色子条 | 工具调用子条 |
| 状态颜色 | 蓝色=完成，绿色=进行中，橙色=等待 | 相同 |
| 缩放控制 | 右下角 - / + 按钮 | 相同 |

## 颜色方案（Vercel 风格）

| 用途 | 颜色 | Hex |
|------|------|-----|
| 背景 | 纯黑 | `#000000` |
| 表面 | 深灰 | `#111111` |
| 边框 | 灰 | `#1f2937` |
| 完成 | 蓝色 | `#3b82f6` |
| 进行中 | 绿色 | `#10b981` |
| 排队中 | 灰色条纹 | `#4b5563` |
| 等待 | 橙色 | `#f59e0b` |
| 失败 | 红色 | `#ef4444` |
| 工具调用 | 深蓝 | `#1e3a5f` |
| 主文字 | 白色 | `#ffffff` |
| 次要文字 | 灰 | `#9ca3af` |
