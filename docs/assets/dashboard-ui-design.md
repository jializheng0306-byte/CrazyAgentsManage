# 智能体实时监控仪表板 UI 设计参考

## Vercel Workflow SDK 风格

本仪表板设计参考 Vercel Workflow SDK 内置观测性仪表板的 UI 风格和设计元素。

### 参考截图

![Vercel Workflow Dashboard](./assets/vercel-workflow-dashboard.png)

### 设计元素提取

| 元素 | Vercel 实现 | 本仪表板适配 |
|------|-------------|-------------|
| **背景色** | 纯黑 (#000) | 相同，保持深色主题一致性 |
| **顶部状态栏** | 白色文字 + 彩色状态圆点 | 智能体总览统计 |
| **标签页** | Trace / Events / Streams | Trace / Events（简化版） |
| **时间轴** | 水平刻度条，带时间标记 | 相同，支持缩放 |
| **任务条** | 彩色横条，显示任务名称和耗时 | 智能体任务跨度条 |
| **嵌套层级** | 缩进 + 连线 | 父子任务缩进树 |
| **状态颜色** | 蓝色=运行中，绿色=完成，橙色=等待 | 相同 |
| **工具调用** | 小圆点/短条 | 工具调用子跨度条 |
| **搜索框** | 顶部搜索栏 | 相同 |
| **缩放控制** | 右下角 +/- 按钮 | 相同 |

### 颜色方案

| 状态 | 颜色 | Hex | 用途 |
|------|------|-----|------|
| Completed | 绿色 | #10b981 | 任务完成 |
| Executing | 蓝色 | #3b82f6 | 任务执行中 |
| Queued | 灰色 | #6b7280 | 任务排队中 |
| Waiting | 橙色 | #f59e0b | 等待依赖 |
| Failed | 红色 | #ef4444 | 任务失败 |
| 背景 | 黑色 | #000000 | 主背景 |
| 边框 | 深灰 | #1f2937 | 卡片边框 |
| 文字 | 白色 | #ffffff | 主文字 |
| 次要文字 | 灰色 | #9ca3af | 次要信息 |

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  Workflow Header (工作流名称、状态、元数据)                   │
├─────────────────────────────────────────────────────────────┤
│  Tabs: [Trace] [Events]                                     │
├─────────────────────────────────────────────────────────────┤
│  Timeline (时间轴刻度 + 任务条)                              │
│  0s ─── 10s ─── 20s ─── 30s ─── 1m ─── 2m                  │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Task 1 [Completed] ──────────────────── 1m 48s      │    │
│  │   └─ Subtask 1.1 [Completed] ─────── 45s            │    │
│  │   └─ Subtask 1.2 [Failed] ──────────── 22.5s        │    │
│  └─────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────┤
│  Controls: [Search] [+/-]                                   │
└─────────────────────────────────────────────────────────────┘
```

### 交互设计

1. **时间线缩放**：点击 +/- 按钮或鼠标滚轮缩放时间轴
2. **任务详情**：鼠标悬停任务条显示 tooltip（目标、角色、耗时、输出预览）
3. **点击跳转**：点击任务条跳转到详情页面
4. **自动刷新**：5 秒轮询更新，右上角显示刷新状态
5. **筛选器**：按来源/角色/状态筛选时间线显示

### 响应式设计

| 屏幕宽度 | 布局 |
|----------|------|
| > 1200px | 完整三列布局（概览 + 时间线 + 事件流） |
| 768-1200px | 双列布局（概览 + 时间线，事件流折叠） |
| < 768px | 单列布局，时间线可水平滚动 |
